# src

La cartella `src/` contiene materiali tecnici riutilizzabili per UD09.

In questa versione non vengono distribuiti script Bash automatici. La generazione delle evidenze e l'esecuzione delle query sono guidate dai file Markdown e dai file KQL commentati.

La sottocartella `src/kql/` contiene query KQL autonome e commentate. Ogni file può essere aperto, copiato nel Logs editor del Portale Azure ed eseguito.

# Evidence UD11

In questa cartella il partecipante può salvare screenshot o note a supporto delle attività UD11.

Esempi di evidenza ammessi:

- schermata della home dell'organizzazione Azure DevOps;
- schermata del progetto Azure DevOps creato;
- schermata della sezione Pipelines;
- schermata della sezione Project settings;
- file compilati nella cartella `docs/`.

Non salvare password, token, chiavi o connection string.

# UD29 — Raccordo verso UD30

## Che cosa sappiamo fare ora

```text
dati validati
→ feature + target
→ training
→ modello
→ prediction
→ valutazione
```

Abbiamo anche imparato che:

```text
prediction ≠ verità
modello complesso ≠ automaticamente migliore
modello vede soltanto le feature fornite
```

## Il prossimo salto

Il Decision Tree produce una classificazione strutturata:

```text
normal / anomaly
```

Un sistema di AI generativa può invece produrre:

```text
testo
sintesi
ipotesi
spiegazioni
```

Questo introduce una nuova domanda:

> Una risposta linguisticamente convincente è necessariamente supportata dalle evidenze?

UD30 partirà da qui:

```text
evidence packet
→ AI
→ risposta
→ verifica umana
```

Poi affronterà quali segnali osservare quando il sistema stesso contiene componenti AI.

# App base UD13

Questa cartella contiene la base applicativa per il laboratorio UD13.

Nel laboratorio guidato il partecipante può copiare questi file in `work/UD13/` oppure crearli passo passo.

Struttura attesa nel repository del partecipante:

```text
work/UD13/
├── src/app.py
├── requirements.txt
├── Dockerfile
├── .dockerignore
└── azure-pipelines.yml
```

# Sorgenti UD08

Questa cartella contiene solo query KQL di esempio.

La versione v5 della UD08 non usa script Bash automatici per eseguire le attività principali. Le operazioni sono descritte passo dopo passo nei file Markdown del laboratorio.

Struttura:

```text
src/
└── kql/
    ├── 01_datatable_take.kql
    ├── 02_datatable_filter_project.kql
    ├── 03_datatable_summarize.kql
    ├── 04_usage_summary.kql
    └── 05_azureactivity_recent.kql
```

Ogni file `.kql` contiene commenti che spiegano scopo, limiti e operatori usati.

# Query KQL UD08

Questa cartella separa le query per origine dei dati.

```text
src/kql/local/   -> query con dataset didattici basati su datatable()
src/kql/azure/   -> query su tabelle reali del Log Analytics Workspace
```

Le query locali servono a imparare la sintassi KQL in modo controllato.
Le query Azure servono a verificare i dati prodotti dalle risorse create nelle UD precedenti.
